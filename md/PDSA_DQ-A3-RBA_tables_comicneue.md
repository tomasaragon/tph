|       |  2  | Decide/Define (problem)             | Define          | ↓                              |
|       |  3  | Diagnose/Discover (root cause)      | Ideate          |                                |

|       |  5  | Predict (effectiveness)             | ↓               | ↓                              |

## PDSA decision making using decision quality (DQ) requirements

| PDSA  | *n* | Requirements | Key questions                      | Description                             |
|:------|:---:|:-------------|:-----------------------------------|:----------------------------------------|
| Plan  |  1  | Frame        | What are we really deciding?       | objectives, perspectives, scope,        |
|       |     |              |                                    | premortem, backcasting                  |
|       |  2  | Values       | What do we need/want?              | values, possible future states,         |
|       |     |              |                                    | must-haves, want-to-haves, trade-offs,  |
|       |  3  | Alternatives | What can we do?                    | creative and doable alternatives,       |
|       |     |              |                                    | activities to start, activities to stop |
|       |  4  | Information  | What do we need to know?           | reliable, relevant information          |
|       |  5  | Reasoning    | Are we thinking straight?          | sound reasoning and reflection,         |
|       |     |              |                                    | multi-objective vs. multi-attribute     |
|       |  6  | Commitment   | Will we do it?                     | commitment to action                    |
| Do    |     | Action plan  | How will we implement plan?        | predict/experiment                      |
| Study |     | Evaluation   | How will we measure effectiveness? | learn                                   |
| Act   |     | Improvement  | Will we adopt, adapt, or abandon?  | improve                                 |



## PDSA Lean A3 Thinking/Reporting

| PDSA  | *n* | Components                | Key questions (order can vary)                        | Notes          |
|:------|:---:|:--------------------------|:------------------------------------------------------|:---------------|
| Plan  |  1  | Background                | What is situation? Why does it matter? What is scope? | context        |
|       |  2  | Problem statement         | What is the need/gap? (not causes or solutions)       | find/decide    |
|       |  3  | Current condition         | What is the current state?                            | context        |
|       |  4  | Objective and Key Results | What are goals and measures of progress/success?      | future states  |
|       |  5  | Analysis                  | What are gaps, root causes, and theory of change?     | reasoning      |
|       |  6  | Countermeasures           | How will CMs activate theory of change (strategies)?  | decide/predict |
| Do    |     | Action plan               | How will we implement plan?                           | experiment     |
| Study |     | Evaluation                | How will we measure effectiveness?                    | learn          |
| Act   |     | Improvement               | Will we adopt, adapt, or abandon?                     | improve        |



## PDSA Results-Based Accountability (RBA)

| PDSA  | *n* | RBA questions                                                  | Notes                                   |
|:------|:---:|:---------------------------------------------------------------|:----------------------------------------|
| Plan  |  1  | What is the “end”?                                             | Objectives (goals)                      |
|       |  2  | How are we doing?                                              | Key Results, incl. trends and forecasts |
|       |  3  | What is the Story Behind the Curve of the baseline?            | Root causes                             |
|       |  4  | Who are partners who have a role to play in Turning the Curve? | “Collective Impact”                     |
|       |  5  | What works to Turn the Curve?                                  | Theory of Change = Strategies           |
|       |  6  | What do you propose to do to Turn the Curve?                   | Countermeasures                         |
| Do    |     | Action plan                                                    | How will we implement plan?             |
| Study |     | Evaluation                                                     | How will we measure effectiveness?      |
| Act   |     | Improvement                                                    | Will we adopt, adapt, or abandon?       |
